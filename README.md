# Simulador de Nóminas con Streamlit

Aplicación interactiva desarrollada en **Python y Streamlit** para simular y analizar los principales elementos de una nómina en España.

La herramienta permite modificar distintos parámetros económicos y visualizar cómo afectan al **salario bruto, cotizaciones a la Seguridad Social y salario neto**.

El proyecto se ha desarrollado como **herramienta docente** para apoyar la enseñanza de conceptos de gestión de personas y costes laborales en asignaturas de empresa e ingeniería.

---

## Objetivo docente

Esta aplicación permite a los estudiantes explorar de forma interactiva:

* estructura de una nómina
* relación entre salario bruto y salario neto
* cotizaciones a la Seguridad Social
* impacto de distintos conceptos retributivos
* simulación de escenarios salariales

El objetivo es facilitar la comprensión de los **costes laborales y su estructura económica**.

## Instalación

Clonar el repositorio:

```
git clone https://github.com/TUUSUARIO/simulador-nominas.git
cd simulador-nominas
```

Instalar dependencias:

```
pip install -r requirements.txt
```

---

## Ejecución de la aplicación

Ejecutar:

```
streamlit run app.py
```

La aplicación se abrirá automáticamente en el navegador.

---

## Uso en el aula

La aplicación puede utilizarse para realizar actividades prácticas como:

* análisis de una nómina real
* simulación de distintos niveles salariales
* impacto de retribución flexible
* comparación entre salario bruto y coste empresa

Esto permite transformar el cálculo de nóminas en **una actividad interactiva de simulación económica**.
